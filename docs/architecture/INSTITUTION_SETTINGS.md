# SecretáriaPay - Configurações por instituição

Este módulo prepara o SecretáriaPay para operar como plataforma multiuniversidade.

Cada instituição pode ter configurações próprias:

- slug público;
- WhatsApp oficial;
- email de suporte;
- país e moeda;
- fuso horário;
- plano de assinatura;
- status de assinatura;
- URL do portal académico;
- regra de bloqueio académico;
- regra de desbloqueio automático;
- dias de tolerância;
- dia padrão de vencimento mensal.

## Exemplo

IMETRO:

- publicSlug: `imetro`
- country: `AO`
- currency: `AOA`
- timezone: `Africa/Luanda`
- subscriptionPlan: `PROFESSIONAL`
- subscriptionStatus: `ACTIVE`
