Fase 14B — Acionar pagamento automático mock a partir do WhatsApp

Objetivo:
- Fazer o teste real pelo WhatsApp, sem precisar chamar o endpoint mock no terminal.
- O aluno recebe a guia, depois envia uma mensagem de teste com o código da cobrança.
- O SecretáriaPay confirma o pagamento mock automaticamente, emite recibo e envia PDF no WhatsApp.

Comandos de teste pelo WhatsApp:
- simular pagamento multicaixa CHG...
- teste pagamento automatico multicaixa CHG...
- pagamento mock iban mesmo banco CHG...
- pagamento mock iban outro banco CHG...
- pagamento mock deposito CHG...
- pagamento mock unitel money CHG...
- pagamento mock afrimoney CHG...

Regra de segurança:
- Só aciona quando a mensagem contém palavra de teste/mock/simular/automático e também contém código CHG...
- Uma mensagem normal como "paguei" NÃO marca pagamento como confirmado automaticamente.
