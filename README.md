# VaiRápido API

Backend do projeto VaiRápido.

## Objetivo

Sistema para compra rápida de passagens pelo WhatsApp, com pagamento via Pix, emissão de bilhete digital e QR Code.

## Stack

- Java 21
- Spring Boot
- PostgreSQL
- Maven
