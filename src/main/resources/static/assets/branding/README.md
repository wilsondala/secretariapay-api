# Marcas oficiais dos documentos

Os arquivos desta pasta correspondem às artes originais aprovadas pelo cliente para uso exclusivo nos documentos emitidos pelo SecretáriaPay Académico.

- `imetro-um-original.png`: monograma UM com louros dourados.
- `secretariapay-metro-original.png`: marca vertical SecretariaPay IMETRO.

As imagens devem ser utilizadas preservando proporção, transparência, cores e composição original. Não redesenhar, substituir tipografia ou distorcer os arquivos.
